Mode d'emploi simplifié

Ce site enregistre vos heures et vos tickets carburant directement sur votre appareil (aucune connexion nécessaire). Voici comment procéder :

  • Onglets (menu horizontal ou ☰ sur mobile) : choisissez un jour, la page carburant ou la prévisualisation.
  • Remplissage : saisissez vos horaires, véhicules, clients... Pas besoin de sauvegarder, tout est mémorisé automatiquement.
  • Exporter / Importer : sauvegardez vos données dans un fichier ou rechargez une semaine précédente. Important : n'oubliez pas d'exporter vos données avant de fermer la page pour pouvoir les réimporter le jour suivant.
  • Réinitialiser : efface tout pour recommencer une semaine propre.
  • Imprimer la semaine : génère le PDF A4 paysage du planning. Imprimer les tickets : produit la feuille carburant.
  • Aperçu des tickets : cliquez sur le bouton dédié pour voir/masquer les photos que vous avez importées.

Organisation des fichiers

Avant de fermer la page, cliquez sur Enregistrer vos données pour récupérer un fichier JSON. Rangez-le ensuite dans le dossier fourni ci-dessous afin de retrouver facilement vos plannings et tickets carburant le lendemain.

1. Téléchargez la structure de dossier recommandée.
2. Placez vos fichiers exportés dans FrigoEst/Plannings et vos tickets carburant dans FrigoEst/TicketsCarburant.
3. Archivez les semaines précédentes dans FrigoEst/Archives si besoin.
4. Le lendemain, utilisez Charger votre planning pour sélectionner le fichier JSON stocké dans le dossier.

Télécharger le dossier FrigoEst (ZIP)

Conseil : après chaque export, ouvrez vos téléchargements et déplacez manuellement le fichier dans le dossier FrigoEst approprié.
